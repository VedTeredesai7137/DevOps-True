print("Hello from Docker - DevOps Experiment 6")

---

## 📦 STEP 3: Create Dockerfile

Create file `Dockerfile` (no extension)

```dockerfile
# Use official Python image
FROM python:3.11-slim

# Set working directory
WORKDIR /app

# Copy file
COPY app.py .

# Run program
CMD ["python", "app.py"]
```

---

## 🏗️ STEP 4: Build Docker Image

```bat
docker build -t devops6:v1 .
```

---

## 🏷️ STEP 5: Tag Image for Docker Hub

```bat

```

# 🔵 PART 2: PYTHON WEB APP + NGINX + DOCKER COMPOSE

Now this is what actually shows **real DevOps understanding**.

---

## 🧾 STEP 1: Create Folder Structure

```bat
mkdir web-app
cd web-app
mkdir app
```

---

## 🐍 STEP 2: Create Flask App

File: `app/app.py`

```python
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Hello from Flask App (Docker + Nginx)"

if __name__ == "__main__":
    app.run(host="0.0.0.0", port=5000)
```

---

## 📦 STEP 3: requirements.txt

```text
flask
```

---

## 📦 STEP 4: Dockerfile (Flask App)

File: `app/Dockerfile`

```dockerfile
FROM python:3.11-slim

WORKDIR /app

COPY . .

RUN pip install -r requirements.txt

CMD ["python", "app.py"]
```

---

## 🌐 STEP 5: Nginx Config

Create file: `nginx.conf`

```nginx
events {}

http {
    server {
        listen 80;

        location / {
            proxy_pass http://app:5000;
        }
    }
}
```

---

## 🧩 STEP 6: docker-compose.yml

Create file:

```yaml
version: '3'

services:

  app:
    build: ./app
    container_name: flask_app
    ports:
      - "5000:5000"

  nginx:
    image: nginx:latest
    container_name: nginx_server
    ports:
      - "8080:80"
    volumes:
      - ./nginx.conf:/etc/nginx/nginx.conf
    depends_on:
      - app
```

---

## 🏗️ STEP 7: Build & Run (IMPORTANT)

```bat
docker-compose up --build
```

---

## 🌍 STEP 8: Access Web App

Open browser:

```
http://localhost:8080
```

---

## ✅ OUTPUT

```text
Hello from Flask App (Docker + Nginx)
```

---

## 🛑 STOP CONTAINERS

```bat
docker-compose down
```
